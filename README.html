# tg-balaboba

Description

This simple telegram bot, allows tou to use yandex.balaboba in telegram chat

Commands

/help - for help /start - to start Type the beginning of the phrase, to get full text from balaboba

To use

To use this bot you need to register your own bot (@BotFather telegram) and add two variables to your system environment
  • BOT_TOKEN - with your bot token
  • BOT_NAME - with your bot name

Techs:

  • Selenium (as some kind of parser)
  • Java Collections
  • Maven
  • telegramBotsApi
